\doxysubsection{Detailed Description}
Represents the simulation model. 

\doxylink{classModel}{Model} will keep track of all the Systems and Flows added to it. We can add, remove and update Systems and Flows. We also run simulations based on the start\+Time and end\+Time.

Overload for = operator. 

This is an overload for = operator, in the format of a = b, in which a is being assigned as b. This will make a deep copy of both containers( \doxylink{classModelImpl_a9be1dcdb4753e5d78e0c817261297dcf}{flows} and \doxylink{classModelImpl_a767e2054ecaaa8d1401ef2c962adc3e2}{systems}) .
