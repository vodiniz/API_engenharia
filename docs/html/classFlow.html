Detailed Description

Flow interface, which will hide the implementation of methods.

Creating this interface will enable us to hide the API implementation from another functions and will permit us to create a layered approach for the API.

virtual double Flow::equation ( )
pure virtual

A pure virtual equation method which defines the flow behavior.

The equation method will define the mathematical relation beetwen the source and target Systems As we do not know the needed implementation we defined flow as an abstract class Whoever is using the API will need to implement this method with the behavior they want.

Returns
return a double value defined by the implemented equation.
